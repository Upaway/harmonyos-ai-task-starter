# v0.1.0 Baseline

基准开源版本说明草稿，用于 GitHub Release 或对内对齐预期。**不是**产品发布会文案。

## What is included

- **可运行的本地专注闭环**：任务输入 → 时长与确认 → 专注计时 → 单次结果 → 统计与设置页；数据默认落在本机（`Preferences`）。
- **工程可复现**：HarmonyOS NEXT Stage 模型、`entry` 模块结构清晰；根目录提供 `build-profile.json5.template`，签名仅在本地配置。
- **开源维护结构**：README、`docs/`（快速开始、架构、路线图、协作与安全）、`CONTRIBUTING.md`、`SECURITY.md`、`CHANGELOG.md`、MIT License；GitHub Issue/PR 模板。

## What is not included yet

- **AI 任务拆解 / 智能规划等能力**：未作为产品功能交付；路线图中的相关项处于后续探索阶段。
- **云端同步、账号体系、多端协作**。
- **复杂报表与长期数据分析**（当前为当日与近期为主的轻量统计）。

## Known limitations

- 早期版本：**UI 与文档仍在迭代**，可能存在体验粗糙或与真实上架物料不一致之处。
- **基准版本**侧重「可运行 + 可维护」，不等同于已完成应用市场上架所需的全部合规与运营材料。
- 部分脚本（如 `scripts/`）用于开发辅助，**非**用户安装包必需路径。

# 截图目录说明

本目录用于存放 **仓库 README 与文档引用的界面截图**，便于展示「开始做事 / Just Start」的真实界面状态。

## 当前引用（与根目录 README 一致）

| 场景 | 建议文件名 |
|------|------------|
| 首页 / 任务输入 | `home.jpg` |
| 开始专注前确认 | `confirm.jpg` |
| 专注进行中 | `focus.jpg` |
| 统计与设置 | `stats.jpg` |

格式可使用 **JPG** 或 **PNG**；更新截图后若改名，请同步修改 [`README.md`](../../README.md) 中的图片链接。

## 占位约定

若你 fork 后尚未接入真实运行环境：

1. 可暂时保留文件名，替换为当前构建下的截图；或  
2. 在 README 中注释掉图片 Markdown，仅保留文字说明，避免指向不存在文件。

## 注意

- 截图中避免包含 **个人隐私**（通知栏手机号、聊天内容等）。
- 上架素材通常有 **分辨率与安全区** 要求，面向商店时请对照华为应用市场文档，不限于本目录规格。
